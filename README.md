# Library Manager
- Application built with Next.js to help manage a library system, including user authentication and data management
- This project is focused on applying modern full-stack best practices using a real world architecture

## Features

### Planned
- CRUD for books
- CRUD for students
- Book borrowing and return management
- User roles and permissions

## Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- PostgreSQL (cloud database with Supabase)
- Supabase Auth (authentication and session management)
- Prisma ORM (data access and business logic)

## Notes
- This project is being developed as a learning-oriented app to improve knowledge of Next.js, React and full-stack architecture
